    Project maintained by spunkpike Hosted on GitHub Pages — Theme by mattgraham

    This is the general purpose database for the sixth semester for databases course of Kirori Mal College.

    For those who are using #mysql and are lazy enough to not create required relations and populate them with data, this is the pre-configured sql file. They may simply run queries. For those using sql server however, this file isn't much of help to them.

    Follow the steps to install this sql file:

    * Uncompress the zip file that you download
* Go to my computer. And the C drive
* Open xampp folder (or similar) (If you do not have an xampp folder proceed to next step)
* Open mysql/bin and paste the .sql file there.
* Execute the following commands though cmd.exe (You can run cmd.exe from Start -> Run -> cmd.exe. ENTER)

    > cd c:\xampp\mysql\bin
// If You hdid not have a xampp folder in step 3 of above steps:
// run    > cd c:\mysql\bin       instead.
> mysql -u root
// You will now see mysql prompt
mysql> CREATE DATABASE 6sem;
mysql> mysql -u root -p 6sem < kmcollege.sql

    When done you can now change the database and see the tables that already have been created.

    mysql> use 6sem;
mysql> show tables;

    The various tables can be viewed by the following command:

    mysql> describe dependent;
mysql> describe employee;
mysql> describe works_on;
mysql> describe department;
